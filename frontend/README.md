# Frontend (React)

Purpose:
- React UI (map, lot list, lot details)

Setup:
- cd frontend
- npm create vite@latest . -- --template react
- npm install

Run:
- npm run dev  (local dev server)

Build:
- npm run build
- npm run preview (test build)

Notes:
- Do not commit node_modules/
- Do not commit .env (use .env.example)
